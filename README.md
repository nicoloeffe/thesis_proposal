# Robust Market Making via World Models and Optimal Transport

**Nicolò Fiordaligi** — MSc Mathematical Engineering, Politecnico di Torino

## Overview

This repository contains the technical report for a research framework 
for distributionally robust evaluation of market making strategies. 
The framework combines a Transformer-based LOB encoder, an autoregressive 
world model with Gaussian Mixture outputs, and a Wasserstein DRO layer 
that constructs adversarial Bellman backups via dual decomposition.

The core output is the robust value curve V_rob(ε), which quantifies 
strategy degradation under adversarial perturbations of increasing intensity.
